This repository contains Terraform projects demonstrating AWS infrastructure as code.

Projects
- 01-vpc — Custom VPC and subnets
- 02-ec2 — EC2 instance and security groups
- 03-s3 — S3 bucket configuration
- 04-remote-state — S3 backend and DynamoDB state locking
- 05-alb — Application Load Balancer with EC2 user data

All resources were created for learning purposes and later destroyed to avoid costs.
